# wagamamaNX
English translation patch for Nintendo Switch version of `Wagamama High Spec`. based on the PS Vita patch created by Froid_san.

# Important Notes 
* The original Switch version is structurally identical to the PS Vita version. All assets are rendered at 960x540.
* This patch directly utilizes translated assets from the PS Vita version. Thus, all on-screen button prompts will display Vita symbols instead of Nintendo Switch icons.
* This patch only replaces script files and UI images. No other game files are modified.
* The font being a bit off is a [known issue](https://www.reddit.com/r/VitaPiracy/comments/9hgydj/comment/e6d35rv/).
* This project was created strictly to test asset compatibility between the PS Vita patch and the Switch version. There are no plans for future updates, optimization, or bug fixes.

# Installation
1.  Download `atmosphere.7z` from the [Releases](../../releases) page and unpack it.
2.  Copy the `0100C5D01281E000` folder to your SD card at `/atmosphere/contents/`.
3.  Launch the game.
